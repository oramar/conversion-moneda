# Conversor de Monedas 👩‍💻

Esta aplicación de escritorio, desarrollada en **Java**, permite realizar conversiones de divisas en tiempo real consultando una API externa. Es un proyecto enfocado en buenas prácticas de programación, manejo de APIs REST y seguridad.

---

## 🛠️ Características Principales
* **Conversión en tiempo real:** Datos actualizados al instante mediante *ExchangeRate-API*.
* **Interfaz intuitiva:** Construida con **Swing** para una experiencia de usuario amigable.
* **Procesamiento eficiente:** Uso de **Gson** para el análisis de respuestas JSON.
* **Seguridad:** Gestión externa de credenciales mediante archivo `config.env`.

---

## 🚀 Tecnologías
* **Lenguaje:** Java 21
* **Interfaz:** Swing (JFC)
* **Librería JSON:** [Gson](https://github.com/google/gson)
* **Control de versiones:** Git
